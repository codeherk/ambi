<!-- GETTING STARTED -->
## Getting Started

### Prerequisites
* npm
```sh
npm install npm@latest -g
```

### Installation

#### Download from releases
Look under [releases]() and download the appropriate software for your operating system

#### Clone source code

1. Clone the repo
```sh
git clone https://github.com/codeherk/ambi.git
```
2. Install NPM packages
```sh
npm install
```
2. Run application
```sh
npm start
```
